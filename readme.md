# 한국공학대학교 디자인공학부 제 20회 졸업전시 웹사이트 (Tech University of Korea Department of Design Engineering 20th Grad Exhibition)

## 프로젝트 개요

본 사이트는 한국공학대학교 디자인공학부 2025년도 졸업전시 작품들을 온라인 전시 형태로 소개하는 웹 애플리케이션입니다.
오프라인 전시 공간을 보완하여, 관람객 및 학생들이 작품 정보를 웹에서 편하게 열람할 수 있도록 만드는 것을 목표로 합니다.

- 목적
    - 졸업 작품 정보 (이미지, 작품 설명, 프로세스, 영상 등)을 구조화하여 온라인에서 제공
    - 한국공학대학교 2025년도 졸업전시 참여 학생들을 소개
    - 한국공학대학교 2025년도 졸업전시 컨셉 및 영상을 소개
    - 이후 기수 학생들이 전대 졸업작품 참고 및 지도교수별 졸업작품 스타일을 참고할 수 있도록 제공

- 대상 사용자
    - 졸업전시 관람객
    - 재학생 및 후속 기수
    - 지도 교수진

## 주요 기능

- 메인 페이지
    - 한국공학대학교 2025년도 졸업전시 컨셉 소개 / 영상 / 이미지 표시
    - 한국공학대학교 2025년도 졸업전시 오프라인 전시 정보 표시
- 프로젝트 페이지
    - 졸업전시 참여 작품을 카드 형태의 목록으로 제공
    - 작품명, 참여자, 썸네일 이미지 표시
    - 카테고리별 필터링
    - 이름/좋아요/조회수/팀작/개인작 필터링
- 프로젝트 상세 페이지
    - 작품 이름, 설명, 제작 의도, 프로세스, 이미지/영상 등 상세 정보 제공
- 게스트북 페이지
    - 웹사이트에 방문한 관람객 및 이용자들이 자유롭게 방명록을 남길 수 있는 공간
- 쇼룸 페이지
    - 한국공학대학교 2025년도 졸업전시 컨셉 상세 설명 공간
- 반응형 레이아웃
    - PC / 태블릿 / 모바일 환경에서의 화면 최적화

### 학생 작품 및 초상 이미지

상세 페이지에 포함된

- 학생 프로필
- 작품 이미지, 상세 정보 이미지
- 작품 설명 및 학생 개인 이메일/SNS 주소

등은 각 학생 개인의 저작물 및 초상권 대상입니다.

- 해당 콘텐츠는 한국공학대학교 디자인공학부 2025년도 졸업전시의 온라인 소개 및 기록을 위한 용도로만 사용됩니다.
- 학생의 개별 동의 없이, 외부 서비스/프로젝트에서의 재사용을 허용하지 않습니다.
- 특히 학생 얼굴이 식별 가능한 이미지는 초상권 보호 대상이며, 상업적/홍보적 용도로의 활용을 금지합니다.
- 학생 본인의 요청이 있는 경우, 합리적인 범위 내에서 해당 콘텐츠의 수정 또는 비공개 처리를 검토합니다.
